Features

  • Search, download, organise and read scientific papers.
  • Clean, modern user interface.
  • Rich interactive reading experience with Lens reader.
  • Data is transmitted peer-to-peer.
  • You control the datasources you want to use.
  • Collected papers are ready for text-mining analysis, stored in JATS XML format.

Technology

  • Built on electron for easy cross-platform compatibility.
  • Rich interactive reading experience with Lens reader.
  • p2p data sharing using hyperdrive technology developed by the Dat project.
