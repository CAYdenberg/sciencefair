Features

  • Search, collect, read and analyse scientific papers.
  • Clean, modern user interface.
  • Rich interactive reading experience.
  • Distributed data store.
  • Control the datasources you use.
  • Text-mining ready corpus.

Technology

  • Science-focused reader provided by lens.
  • Distributed data sharing using hyperdrive technology.
  • Fast local search index built on searchindex technology.
  • Built using web technologies with choo and electron.

Community

ScienceFair v1 was developed by Code for Science and FathomLabs along with other contributors, with support from eLife and the Dat project.

If you find a bug in ScienceFair please report it on the issue tracker. You can chat with us any time on IRC (#sciencefair on freenode.net).

ScienceFair is an Open Source project. Code is licensed under the MIT license and hosted on Github. We welcome contributions of any kind from anyone, provided you follow our code of conduct.
